# Intro

This is a small tool to manage ssh server info in command line. Write the serve
so that you don't need to remember the servers' ip, password or port.

# Dependency

* ncurse: draw gui
* sshpass: take password form config file

# usage

* j: move down
* k: move up
* r: reload
* q: exit
* G: move to last
* \r: connect
